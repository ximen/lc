# Leakage controller

Firmware for ESP32-based leakage controller (hw see [here](https://easyeda.com/ximen/water))

## Features:
- 6 wired sensors
- 2 drives control
- Relay output
- Embedded web server for configuration 
- Internal alarm buzzer
- Telegram events notification
- MQTT control
